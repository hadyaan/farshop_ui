## Built With
- **Flutter Version**: 3.35.5
- **Dart Version**: 3.9.2
- **State Management**: GetX
- **Persistence**: GetStorage
- **Code Generator**: flutter_gen
- **Custom Theming**: Light & Dark mode
- **Widget Components**: Custom Cards, Banners, Chips, Bottom Navigation

## Cara Menjalankan Project

Pastikan sudah menginstall Flutter di perangkatmu.

1. Clone repository
```bash
git clone https://github.com/USERNAME/farshop.git
```
2. Masuk ke folder project:
```bash
cd farshop
```
3. Install semua depedency:
```bash
flutter pub get
```
4. Jalankan aplikasi:
```bash
flutter run
```
